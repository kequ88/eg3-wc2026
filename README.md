# Elmina Green Three — World Cup 2026 Predictor ⚽

A neighborhood tournament prediction application for EG3 residents, engineered using Vite, Firebase Firestore, and Netlify hosting.

---

## ⚡ Quick Start

```bash
# Clone and enter the repository
git clone [https://github.com/kequ88/eg3-wc2026.git](https://github.com/kequ88/eg3-wc2026.git) && cd eg3-wc2026

# Install dependencies and initialize environment variables
npm install
cp .env.example .env

# Run local development server (localhost:5173)
npm run dev
